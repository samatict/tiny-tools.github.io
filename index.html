Here is a box of useful tiny tools I created, from time to time.

Go to github repo: https://github.com/samatht/tiny-tools.github.io

  • readoutloud-textbox-html-SamAT.html: A simple HTML page that allows users to enter the text into the textarea and let Microsoft Edge browser to read out loud. How to use: download this html file to a computer/phones and open it with Microsoft Edge browser.
  • Khmer Analog Clock: A simple analog clock with date and time in Khmer text.
